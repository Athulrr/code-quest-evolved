## 🌟 **GAME OVERVIEW**

**CodeQuest** is an epic coding adventure that transforms programming challenges into an immersive gaming experience! Battle through increasingly difficult algorithms, defeat code monsters, and become the ultimate programming champion.

---

## 🎮 **FEATURES**

### ⚔️ **BATTLE SYSTEM**
- **Hero vs Monster**: Face off against animated enemies in epic coding battles
- **Custom Characters**: Your hero image vs animated ghost enemies
- **Dramatic Animations**: Slash, hurt, and explode effects during gameplay
- **Defeat Sequences**: Ghost grows to cover screen when you lose

### 🧠 **CHALLENGE PROGRESSION**
- **8 Difficulty Levels**: From beginner to expert algorithms
- **AI-Generated Challenges**: Adaptive difficulty based on performance
- **Classic Mode**: Fixed progression through curated challenges
- **Performance Tracking**: Speed and accuracy affect next challenge

### 🎯 **CODING CHALLENGES**

#### **Level 1-2: Foundation**
- Fibonacci recursion
- Anagram detection

#### **Level 3-4: Intermediate**
- Longest Common Subsequence
- Binary Search implementation

#### **Level 5-6: Advanced**
- Prime factorization
- Merge Sort algorithm

#### **Level 7-8: Expert**
- Longest Palindromic Substring
- Depth-First Search on graphs

### 🎉 **SPECIAL FEATURES**
- **Victory Video**: Anime celebration at level 5 completion
- **8-Bit Aesthetics**: Retro gaming font and styling
- **Performance Metrics**: Track your coding speed and success rate
- **Adaptive AI**: Challenges adjust to your skill level

## 🚀 **GETTING STARTED**

### **Prerequisites**
```bash
Node.js (v16 or higher)
npm or yarn
```

### **Installation**
```bash
# Clone the repository
git clone https://github.com/yourusername/code-quest-evolved.git

# Navigate to project directory
cd code-quest-evolved

# Install dependencies
npm install

# Start the development server
npm run dev
```

### **Game Modes**

#### **🎮 Classic Mode**
- Fixed 8-level progression
- Curated challenges for all skill levels
- Perfect for beginners

#### **🤖 AI Mode**
- Adaptive difficulty based on performance
- Infinite levels with AI-generated challenges
- Requires OpenAI API key

## 🎯 **GAMEPLAY MECHANICS**

### **Challenge Structure**
1. **Problem Description**: Clear explanation of what to implement
2. **Starter Code**: Function skeleton to complete
3. **Test Cases**: Automatic validation of your solution
4. **Performance Tracking**: Time and attempt monitoring

### **Scoring System**
- **Speed Bonus**: Faster completion = higher difficulty
- **Success Rate**: Consistent wins = harder challenges
- **Adaptive Difficulty**: AI adjusts based on performance

### **Victory Conditions**
- ✅ Correct output for test cases
- ✅ Proper function implementation
- ✅ Efficient algorithm design

## 🎨 **CUSTOMIZATION**

### **Adding Your Own Images**
1. Place hero image in `public/hero.png`
2. Update enemy GIF URL in the component
3. Customize animations in `src/index.css`

### **Modifying Challenges**
- Edit `fallbackChallenges` array in `CodeQuest.tsx`
- Add new difficulty levels
- Customize AI prompts for different challenge types

## 🐛 **TROUBLESHOOTING**

### **Common Issues**
```
❌ "Vite not recognized" → Run: npm install
❌ "Module not found" → Check Node.js version (v16+)
❌ "API errors" → Verify OpenAI API key
❌ "Video not playing" → Check browser autoplay settings
```

### **Performance Tips**
- Use modern browsers for best experience
- Enable JavaScript and autoplay
- Stable internet connection for AI features
